### 主要原因及解决方案

运行器标签不匹配

工作流中指定的 runs-on 标签必须与运行器配置的标签完全一致（区分大小写）。
✅ 操作：检查运行器的标签设置（仓库或组织 Settings → Actions → Runners → 编辑标签），确保与 YAML 文件中的 runs-on 匹配。
